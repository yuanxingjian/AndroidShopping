# AndroidShopping
商城APP(已打包APK)                                                                                                      
    本项目是在线商城 APP，主要实现了主页、商品详情页、购物车三个界面。商品数据来源于 Tomcat 服务器中的图片和 Json 文件。
其中主页主要包含广告横幅、频道、活动、秒杀商品、推荐新品、热卖商品以及置顶悬浮按钮。在商品详情界面中，可以将商品添加到购物车；
在购物车界面，可以计算被选中的商品的总价格，以及移除其中的商品。
